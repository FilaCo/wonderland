# Wonderland

## What is Wonderland?

Wonderland is a 2D game engine built in C++.

## ⚠️ WARNING ⚠️

Wonderland is in the early stages of development. Important features are missing. Documentation is sparse.

## Design Goals

- **Focused:** Offer a complete 2D feature set (no 3D here).
- **Fast:** App logic should run quickly and, when possible, in parallel.
- **Specialized:** Alice for game logic, Hatter for UI and AI staff.
